# Taylor Series in C
In this repository you'll find code and executables of C programmed Taylor Series for a lot of mathematical functions and some constants.
1.  $ln(2)$
2.  $\frac{\pi}{4}$
3.  $\frac{\pi^2}{6}$
4.  $\frac{\pi^2}{8}$
5.  $\frac{1}{2}$
6.  $\frac{3}{4}$
7.  $e^x$
8.  $xe^x$
9.  $(x+x^2)e^x$
10. $ln(1+x), -1 < x \leq 1$
11. $\frac{1}{2}ln(\frac{1+x}{1-x}), -1 < x < 1$
12. $ln(x), x > 0$
13. $ln(x), x \geq \frac{1}{2}$
